жЁЎејҸеҚЎзүҮ #4пјҡеҲҶеұӮ Agent зј–жҺ’

жқҘжәҗеӯҗзі»з»ҹпјҡеӨҡжҷәиғҪдҪ“жһ¶жһ„пјҲMulti-Agentпјү Claude Code е…ій”®ж–Үд»¶пјҡsrc/tasks/LocalAgentTask/пјҢsrc/tasks/RemoteAgentTask/пјҢsrc/tasks/InProcessTeammateTask/

й—®йўҳ

еҰӮдҪ•ж №жҚ®д»»еҠЎзү№еҫҒйҖүжӢ©еҗҲйҖӮзҡ„ agent жү§иЎҢжЁЎејҸпјҲжң¬ең°/иҝңзЁӢ/иҝӣзЁӢеҶ…пјүпјҢеңЁжҖ§иғҪгҖҒйҡ”зҰ»жҖ§гҖҒеҚҸдҪңиғҪеҠӣд№Ӣй—ҙеҸ–еҫ—е№іиЎЎпјҹ

ж–№жЎҲ

дёүз§Қ Task зұ»еһӢй’ҲеҜ№жң¬иҙЁдёҚеҗҢзҡ„йңҖжұӮпјҡ

LocalAgentTaskпјҲsrc/tasks/LocalAgentTask/пјүпјҡ

  • йҖҡиҝҮ createSubagentContext() еҲӣе»әйҡ”зҰ»зҡ„ ToolUseContextпјҲе…ӢйҡҶ readFileStateпјҢзӢ¬з«Ӣ AbortControllerпјҢno-op setAppStateпјү
  • и°ғз”Ё runAgent() вҶ’ query() еңЁеҗҢдёҖ JS зәҝзЁӢеҶ…иҝҗиЎҢе®Ңж•ҙзҡ„ agent еҫӘзҺҜ
  • з»“жһңйҖҡиҝҮ enqueuePendingNotification() еҶҷе…Ҙж¶ҲжҒҜйҳҹеҲ—пјҢд»Ҙ XML user-role ж¶ҲжҒҜеҪўејҸжіЁе…ҘзҲ¶еҜ№иҜқ
  • еҸ–ж¶ҲпјҡabortController.abort() дёӯж–ӯ query еҫӘзҺҜпјҢз«ӢеҚіз”ҹж•Ҳ

RemoteAgentTaskпјҲsrc/tasks/RemoteAgentTask/пјүпјҡ

  • йҖҡиҝҮ teleport API еңЁ CCR дә‘з«ҜеҲӣе»ә sessionпјҢжң¬ең°еҸӘдҝқеӯҳ sessionId
  • жҜҸз§’йҖҡиҝҮ pollRemoteSessionEvents() жӢүеҸ–еўһйҮҸ SDKMessage[]
  • session metadata еҶҷе…ҘзЈҒзӣҳ sidecarпјҲ--resume ж—¶еҸҜжҒўеӨҚпјү
  • з»“жһңйҖҡиҝҮдё“з”Ёж Үзӯҫи§ЈжһҗпјҲ<remote-review>гҖҒ<ultraplan> зӯүпјүжҲ– result ж¶ҲжҒҜзұ»еһӢеҲӨж–ӯ

InProcessTeammateTaskпјҲsrc/tasks/InProcessTeammateTask/пјүпјҡ

  • йҖҡиҝҮ runWithTeammateContext() жіЁе…Ҙ AsyncLocalStorageпјҢе®һзҺ°дёҠдёӢж–Үйҡ”зҰ»иҖҢйқһиҝӣзЁӢйҡ”зҰ»
  • runInProcessTeammate() е®һзҺ°жҢҒз»ӯиҝҗиЎҢзҡ„ while еҫӘзҺҜпјҢidle еҗҺзӯүеҫ… mailbox ж¶ҲжҒҜ
  • жқғйҷҗйңҖжұӮи·Ҝз”ұеҲ° leader зҡ„ ToolUseConfirmQueueпјҢUI жҳҫзӨә worker badge
  • еҸҢ AbortControllerпјҡlifecycleпјҲжқҖжӯ»ж•ҙдёӘ teammateпјү+ currentWorkпјҲжү“ж–ӯеҪ“ж¬Ўе·ҘдҪңпјү

е…ій”®и®ҫи®ЎеҶізӯ–

дёәд»Җд№ҲйңҖиҰҒдёүз§ҚдёҚеҗҢзҡ„ Task зұ»еһӢиҖҢдёҚжҳҜз»ҹдёҖдёҖз§Қпјҹ

дёүз§Қзұ»еһӢи§ЈеҶізҡ„жҳҜжң¬иҙЁдёҚеҗҢзҡ„й—®йўҳпјҡ

  1. LocalAgentTask жҳҜгҖҢеҚіеҸ‘еҚіеҝҳгҖҚеҗҺеҸ°д»»еҠЎпјҢдёҚйңҖиҰҒй•ҝжңҹеӯҳжҙ»пјҢд№ҹдёҚйңҖиҰҒдёҺе…¶д»– agent еҚҸдҪң
  2. RemoteAgentTask жҳҜгҖҢзҰ»зәҝжү§иЎҢгҖҚд»»еҠЎпјҢжң¬ең°еҸӘжҳҜи§ӮеҜҹиҖ…пјӣдә‘з«ҜжңүзӢ¬з«Ӣзҡ„и®Ўз®—иө„жәҗе’ҢзҪ‘з»ңйҡ”зҰ»пјҢжң¬ең°еҙ©жәғдёҚеҪұе“Қжү§иЎҢ
  3. InProcessTeammateTask жҳҜгҖҢжҢҒз»ӯеңЁеңәгҖҚзҡ„еҚҸдҪң agentпјҢйңҖиҰҒ idle зӯүеҫ…гҖҒжҺҘж”¶ DMгҖҒиҜ·жұӮжқғйҷҗе®Ўжү№гҖҒдёҺеӣўйҳҹе…¶д»–жҲҗе‘ҳе…ұдә«д»»еҠЎеҲ—иЎЁ

йҖүжӢ©ж ҮеҮҶпјҡ

  • еҚ•ж¬ЎгҖҒеҗҺеҸ°гҖҒж— йңҖдәӨдә’ вҶ’ local_agent
  • и¶…й•ҝж—¶й—ҙгҖҒдә‘зҺҜеўғгҖҒйңҖиҰҒ CCR вҶ’ remote_agent
  • еӨҡ agent еӣўйҳҹгҖҒйңҖиҰҒжҢҒз»ӯеҚҸдҪңгҖҒйңҖиҰҒжқғйҷҗ UI вҶ’ in_process_teammate

з»“жһңејӮжӯҘеҢ–жҳҜз»ҹдёҖжңәеҲ¶пјҡж— и®әжң¬ең°иҝҳжҳҜиҝңзЁӢпјҢз»“жһңйғҪйҖҡиҝҮ enqueuePendingNotification() жіЁе…Ҙж¶ҲжҒҜйҳҹеҲ—пјҢзҲ¶ agent ж°ёиҝңдёҚгҖҢзӯүеҫ…гҖҚеӯҗд»»еҠЎпјҢиҖҢжҳҜгҖҢжҺҘж”¶йҖҡзҹҘгҖҚгҖӮ

йҖӮз”ЁжқЎд»¶

  • зі»з»ҹйңҖиҰҒж”ҜжҢҒеӨҡз§Қ agent жү§иЎҢеңәжҷҜпјҲеҚ•ж¬ЎеҗҺеҸ°д»»еҠЎгҖҒй•ҝж—¶й—ҙдә‘д»»еҠЎгҖҒжҢҒз»ӯеҚҸдҪңеӣўйҳҹпјү
  • зҲ¶еӯҗ agent д№Ӣй—ҙйңҖиҰҒејӮжӯҘйҖҡдҝЎпјҲдёҚйҳ»еЎһзҲ¶ agent з»§з»ӯеӨ„зҗҶе…¶д»–дәӢеҠЎпјү
  • йңҖиҰҒе·®ејӮеҢ–зҡ„йҡ”зҰ»зә§еҲ«пјҲиҝӣзЁӢеҶ…йҡ”зҰ» vs дә‘з«Ҝйҡ”зҰ»пјү

жқғиЎЎ

дјҳзӮ№пјҡ

  • еҗ„зұ»еһӢй’ҲеҜ№еңәжҷҜдјҳеҢ–пјҢLocalAgent еҗҜеҠЁејҖй”ҖдҪҺпјҢRemoteAgent жң¬ең°еҙ©жәғдёҚеҪұе“Қ
  • з»ҹдёҖзҡ„ејӮжӯҘйҖҡзҹҘжңәеҲ¶пјҢзҲ¶ agent д»Јз ҒдёҚеҢәеҲҶеӯҗд»»еҠЎзұ»еһӢ
  • AbortController ж ‘з»“жһ„пјҢзҲ¶ abort иҮӘеҠЁдј ж’ӯз»ҷеӯҗ agent

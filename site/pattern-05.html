жЁЎејҸеҚЎзүҮ #5пјҡCoordinator-Worker жЁЎејҸ

жқҘжәҗеӯҗзі»з»ҹпјҡеӨҡжҷәиғҪдҪ“жһ¶жһ„пјҲMulti-Agentпјү Claude Code е…ій”®ж–Үд»¶пјҡsrc/coordinatorMode.tsпјҢsrc/tasks/LocalAgentTask/пјҢsrc/tools/AgentTool/

й—®йўҳ

еҰӮдҪ•е°ҶеӨ§д»»еҠЎеҲҶи§Јз»ҷеӨҡдёӘе№¶иЎҢ agent жү§иЎҢпјҢе№¶иҒҡеҗҲз»“жһңпјҢеҗҢж—¶йҒҝе…Қ coordinator иҮӘиә«жү§иЎҢе…·дҪ“ж“ҚдҪңеёҰжқҘзҡ„ж··д№ұпјҹ

ж–№жЎҲ

Coordinator йҖҡиҝҮзҺҜеўғеҸҳйҮҸ CLAUDE_CODE_COORDINATOR_MODE=1 жҝҖжҙ»пјҢжҺҘж”¶дё“з”Ё system promptпјҲgetCoordinatorSystemPrompt()пјүпјҡ

е·Ҙе…·йӣҶдёҘж јйҷҗе®ҡпјҡCoordinator еҸӘжӢҘжңү {AgentTool, TaskStopTool, SendMessageTool, SyntheticOutputTool}пјҢж— жі•зӣҙжҺҘжү§иЎҢж–Үд»¶ж“ҚдҪңжҲ– bash е‘Ҫд»ӨгҖӮ

Worker е·Ҙе…·йӣҶжіЁе…ҘпјҡйҖҡиҝҮ getCoordinatorUserContext() е°Ҷ worker зҡ„е·Ҙе…·йӣҶдҝЎжҒҜжіЁе…Ҙ coordinator зҡ„ user contextпјҢи®© coordinator зҹҘйҒ“ worker иғҪеҒҡд»Җд№ҲпјҢеҸҜд»ҘеңЁ prompt дёӯжҢҮеҜј worker дҪҝз”Ёзү№е®ҡе·Ҙе…·гҖӮ

е·ҘдҪңжөҒеӣӣйҳ¶ж®өпјҡResearch вҶ’ SynthesisпјҲcoordinator дәІиҮӘе®ҢжҲҗпјүвҶ’ Implementation вҶ’ VerificationгҖӮSynthesis йҳ¶ж®өз”ұ coordinator иҮӘе·ұе®ҢжҲҗпјҢзҰҒжӯўеҶҷ"еҹәдәҺдҪ зҡ„еҸ‘зҺ°"иҝҷзұ»жҮ’жҖ§е§”жҙҫгҖӮ

е®Ңе…ЁејӮжӯҘж¶ҲжҒҜжөҒпјҡ

Coordinator
    |-- AgentTool(prompt) в”Җв”Җ> Worker (LocalAgentTask)
    |                              |
    |                         runAgent() / enqueuePendingNotification()
    |<в”Җв”Җ user-role <task-notification> XML в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ|
    |
    |-- SendMessageTool(to=agentId, msg) в”Җв”Җв”Җв”Җв”Җ> еҗҢдёҖ WorkerпјҲеӨҚз”ЁдёҠдёӢж–Үпјү

з»§з»ӯ vs ж–°е»ә Worker зҡ„еҶізӯ–зҹ©йҳөпјҡ

  • з ”з©¶иҝҮзҡ„ж–Үд»¶йңҖиҰҒзј–иҫ‘ вҶ’ SendMessageTool continueпјҲworker е·Іжңүж–Үд»¶еңЁ contextпјү
  • з ”з©¶иҢғеӣҙе®ҪдҪҶе®һзҺ°иҢғеӣҙзӘ„ вҶ’ AgentTool ж–°е»әпјҲйҒҝе…ҚжҗәеёҰеҷӘеЈ°дёҠдёӢж–Үпјү
  • дҝ®еӨҚе·Іжңү worker зҡ„еӨұиҙҘ вҶ’ continueпјҲworker жңүй”ҷиҜҜдёҠдёӢж–Үпјү
  • зӢ¬з«ӢйӘҢиҜҒ вҶ’ ж–°е»әпјҲйӘҢиҜҒиҖ…еә”жңүж–°йІңи§Ҷи§’пјү

е…ій”®и®ҫи®ЎеҶізӯ–

  1. е·Ҙе…·иҝҮж»ӨеҚіи§’иүІе®ҡд№үпјҡcoordinator еҸӘжңүзј–жҺ’е·Ҙе…·пјҢдёҚиғҪзӣҙжҺҘжү§иЎҢгҖӮиҝҷдёҚжҳҜжҠҖжңҜйҷҗеҲ¶пјҢиҖҢжҳҜи§’иүІиҒҢиҙЈзҡ„д»Јз ҒеҢ–иЎЁиҫҫпјҡ

    • йҳІжӯў coordinator еҲҶеҝғпјҡзҰҒжӯўзӣҙжҺҘе·Ҙе…·и°ғз”Ёиҝ«дҪҝе®ғжҠҠжүҖжңүе·ҘдҪңе§”жҙҫеҮәеҺ»пјҢиҮӘе·ұдё“жіЁдәҺе…ЁеұҖи§Ҷи§’
    • ејәеҲ¶е№¶иЎҢеҢ–пјҡеӨҡдёӘзӢ¬з«Ӣзҡ„ AgentTool и°ғз”ЁеҸҜд»ҘеңЁеҗҢдёҖж¶ҲжҒҜдёӯе№¶еҸ‘еҸ‘еҮәпјҢжҳҜе№¶иЎҢеҢ–зҡ„еӨ©з„¶и§ҰеҸ‘зӮ№
    • е®үе…Ёйҡ”зҰ»пјҡcoordinator ж— жі•зӣҙжҺҘдҝ®ж”№ж–Үд»¶зі»з»ҹпјҢеҚідҪҝ coordinator дә§з”ҹе№»и§үпјҢд№ҹж— жі•зӣҙжҺҘйҖ жҲҗд»Јз Ғз ҙеқҸ

  2. Synthesis йҳ¶ж®өеҝ…йЎ»з”ұ coordinator дәІиҮӘе®ҢжҲҗпјҡзҰҒжӯўжҠҠдҝЎжҒҜеҗҲжҲҗе§”жҙҫз»ҷ workerпјҢзЎ®дҝқ coordinator зңҹжӯЈзҗҶи§Јеҗ„ worker зҡ„еҸ‘зҺ°пјҢиҖҢдёҚд»…д»…жҳҜиҪ¬еҸ‘гҖӮ

йҖӮз”ЁжқЎд»¶

  • д»»еҠЎеҸҜд»ҘеҲҶи§ЈдёәеӨҡдёӘзӣёеҜ№зӢ¬з«Ӣзҡ„еӯҗд»»еҠЎпјҢдё”еӯҗд»»еҠЎд№Ӣй—ҙдёҚйңҖиҰҒе®һж—¶йҖҡдҝЎ
  • йңҖиҰҒе№¶иЎҢеҠ йҖҹпјҲеҗҢж—¶еҗҜеҠЁеӨҡдёӘ Research workerпјү
  • coordinator зҡ„дё»иҰҒд»·еҖјеңЁдәҺд»»еҠЎеҲҶи§Је’Ңз»“жһңж•ҙеҗҲпјҢиҖҢйқһе…·дҪ“жү§иЎҢ

жқғиЎЎ

дјҳзӮ№пјҡ

  • е№¶иЎҢз ”з©¶/е®һзҺ°еӨ§е№…жҸҗйҖҹ
  • жё…жҷ°зҡ„д»»еҠЎеҪ’еұһпјҲи°ҒеҒҡдәҶд»Җд№Ҳжңү task-id еҸҜиҝҪжәҜпјү
  • coordinator дёҠдёӢж–Үдё“жіЁдәҺй«ҳеұӮеҶізӯ–
  • еҸҜд»ҘеҒңжӯўй”ҷиҜҜж–№еҗ‘зҡ„ workerпјҲTaskStopToolпјү

зјәзӮ№/еұҖйҷҗпјҡ

  • coordinator иҪ®ж¬ЎеўһеҠ пјҲжҜҸж¬Ўзӯү worker еӣһжҠҘжүҚиғҪз»§з»ӯпјү
  • worker дёҚиғҪзңӢеҲ°еҪјжӯӨзҡ„е·ҘдҪңпјҲеҸӘиғҪйҖҡиҝҮ coordinator дёӯиҪ¬пјү
  • й•ҝе№¶иЎҢ session дјҡз§ҜзҙҜеӨ§йҮҸ task-notification ж¶ҲжҒҜпјҢеўһеҠ  context й•ҝеәҰ
  • worker д№Ӣй—ҙж— жі•зӣҙжҺҘеҚҸи°ғпјҲж—  peer-to-peer йҖҡдҝЎпјү

еңЁиҮӘе·ұйЎ№зӣ®дёӯзҡ„еә”з”ЁжҖқиҖғ

иҝҷдёӘй—®йўҳеңЁжҲ‘зҡ„йЎ№зӣ®дёӯеӯҳеңЁеҗ—пјҹ пјҲеҫ…еЎ«еҶҷпјү

еҸҜд»ҘзӣҙжҺҘз”Ёеҗ—пјҹиҝҳжҳҜйңҖиҰҒз®ҖеҢ–/и°ғж•ҙпјҹ пјҲеҫ…еЎ«еҶҷпјү

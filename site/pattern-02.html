жЁЎејҸеҚЎзүҮ #2пјҡTool жҺҘеҸЈз»ҹдёҖжҠҪиұЎ

жқҘжәҗеӯҗзі»з»ҹпјҡе·Ҙе…·зі»з»ҹпјҲTool Systemпјү Claude Code е…ій”®ж–Үд»¶пјҡsrc/tools/Tool.tsпјҲL362, L783пјүпјҢsrc/tools/tools.ts

й—®йўҳ

еҰӮдҪ•и®©иғҪеҠӣе·®ејӮжһҒеӨ§зҡ„е·Ҙе…·пјҲж–Үд»¶иҜ»еҸ–гҖҒBash жү§иЎҢгҖҒзҪ‘з»ңиҜ·жұӮгҖҒеӯҗ agent еҗҜеҠЁгҖҒMCP д»ЈзҗҶпјүиў«жЁЎеһӢд»Ҙе®Ңе…Ёз»ҹдёҖзҡ„ж–№ејҸи°ғз”ЁпјҢдё”иғҪеңЁжқғйҷҗзі»з»ҹгҖҒUIгҖҒеәҸеҲ—еҢ–еҗ„еұӮж¬Ўж— зјқеӨ„зҗҶпјҹ

ж–№жЎҲ

еҚ•дёҖ Tool<Input, Output, Progress> жіӣеһӢжҺҘеҸЈпјҲTool.ts:362пјүпјҡзұ»еһӢеҸӮж•°и®©жҜҸдёӘе·Ҙе…·зҡ„иҫ“е…Ҙ/иҫ“еҮә/иҝӣеәҰйғҪжңүзІҫзЎ®зұ»еһӢпјҢдҪҶи°ғз”ЁжЎҶжһ¶еҸҜд»Ҙз”Ё Tool<AnyObject, unknown> з»ҹдёҖжҢҒжңүгҖӮжҺҘеҸЈзәҰ 40 дёӘжҲҗе‘ҳпјҢж¶өзӣ–жү§иЎҢгҖҒжқғйҷҗгҖҒAPI еәҸеҲ—еҢ–гҖҒUI жёІжҹ“гҖҒи·Ҝеҫ„дҝЎжҒҜдә”дёӘз»ҙеәҰпјҡ

  • жү§иЎҢж ёеҝғпјҡcall(input, ctx, onProgress) вҶ’ Promise<ToolResult>пјҢisEnabled()пјҢisConcurrencySafe()пјҢisReadOnly()пјҢisDestructive()
  • жқғйҷҗпјҡvalidateInput()пјҢcheckPermissions()пјҢpreparePermissionMatcher()
  • API еәҸеҲ—еҢ–пјҡmapToolResultToToolResultBlockParam()пјҢinputSchemaпјҲZod schema вҶ’ JSON Schemaпјү
  • UI жёІжҹ“пјҡrenderToolUseMessage()пјҢrenderToolResultMessage()пјҢrenderToolUseProgressMessage() зӯү
  • и·Ҝеҫ„дҝЎжҒҜпјҡgetPath()пјҢinputsEquivalent()

buildTool(def: ToolDef) вҶ’ Tool е·ҘеҺӮеҮҪж•°пјҲTool.ts:783пјүпјҡжҺҘеҸ—зңҒз•ҘдәҶеҸҜй»ҳи®Өж–№жі•зҡ„ ToolDefпјҢеЎ«е…Ҙе®үе…Ёй»ҳи®ӨеҖјеҗҺиҝ”еӣһе®Ңж•ҙ ToolгҖӮжүҖжңүе·Ҙе…·йҖҡиҝҮжӯӨеҮҪж•°жһ„е»әпјҢдҝқиҜҒжҺҘеҸЈе®Ңж•ҙжҖ§гҖӮй»ҳи®ӨеҖјеқҮдёә fail-closedпјҡisConcurrencySafe=falseпјҢisReadOnly=falseпјҢcheckPermissions=allowгҖӮ

JSON Schema дҪңдёә API иҫ№з•ҢпјҡMCP е·Ҙе…·зӣҙжҺҘжҸҗдҫӣ JSON SchemaпјҢеҶ…зҪ®е·Ҙе…·йҖҡиҝҮ Zod schema иҮӘеҠЁиҪ¬жҚўпјҢдёӨз§Қи·Ҝеҫ„жңҖз»ҲйғҪз”ҹжҲҗж ҮеҮҶ JSON Schema еҸ‘з»ҷ APIгҖӮ

е…ій”®и®ҫи®ЎеҶізӯ–

  1. inputSchema з”Ё Zod иҖҢйқһзәҜ TypeScript зұ»еһӢпјҡZod еҗҢж—¶жҸҗдҫӣиҝҗиЎҢж—¶ж ЎйӘҢпјҲtool.inputSchema.parse(input)пјүгҖҒзұ»еһӢжҺЁж–ӯпјҲz.infer<Input>пјүгҖҒJSON Schema з”ҹжҲҗпјҲеҸ‘з»ҷ APIпјүдёүеҗҲдёҖгҖӮзәҜ TS зұ»еһӢеңЁзј–иҜ‘еҗҺж“ҰйҷӨпјҢж— жі•еңЁиҝҗиЎҢж—¶иҜ»еҸ–жҲ–еәҸеҲ—еҢ–гҖӮ

  2. buildTool() е·ҘеҺӮеҮҪж•°зҡ„й»ҳи®ӨеҖјзӯ–з•Ҙпјҡж–°е·Ҙе…·ж— жі•йҒ—жјҸе…ій”®е®үе…Ёж–№жі•пјҢfail-closed й»ҳи®ӨеҖјдҝқиҜҒдәҶе®үе…Ёеә•зәҝвҖ”вҖ”еҚідҪҝеҝҳи®°е®һзҺ° isReadOnly()пјҢй»ҳи®Өиҝ”еӣһ falseпјҲжҢүз ҙеқҸжҖ§е·Ҙе…·еӨ„зҗҶпјүпјҢдёҚдјҡж„ҸеӨ–и·іиҝҮжқғйҷҗжЈҖжҹҘгҖӮ

  3. mapToolResultToToolResultBlockParam() з”ұе·Ҙе…·иҮӘе·ұе®һзҺ°пјҡжҜҸдёӘе·Ҙе…·иҮӘе·ұзҹҘйҒ“еҰӮдҪ•е°Ҷз»“жһ„еҢ–иҫ“еҮәеәҸеҲ—еҢ–дёә API tool_resultпјҢж–Үжң¬/еӣҫзүҮ/notebook еҗ„жңүе®һзҺ°пјҢжЎҶжһ¶ж— йңҖдәҶи§ЈеҶ…йғЁз»“жһ„гҖӮ

йҖӮз”ЁжқЎд»¶

йңҖиҰҒе°ҶејӮжһ„иғҪеҠӣпјҲж–Үд»¶ I/OгҖҒиҝӣзЁӢжү§иЎҢгҖҒзҪ‘з»ңгҖҒеӯҗзі»з»ҹи°ғз”Ёпјүз»ҹдёҖжҡҙйңІз»ҷжЁЎеһӢ/жЎҶжһ¶/UI ж—¶пјӣе·Ҙе…·ж•°йҮҸи¶…иҝҮ 10 дёӘдё”йңҖиҰҒз»ҹдёҖжқғйҷҗз®ЎзҗҶж—¶гҖӮ

жқғиЎЎ

дјҳзӮ№пјҡ

  • жҺҘеҸЈз»ҹдёҖпјҢжЎҶжһ¶д»Јз ҒпјҲжқғйҷҗгҖҒеәҸеҲ—еҢ–гҖҒUIпјүеҸӘйңҖйқўеҜ№ Tool жҺҘеҸЈпјҢдёҚе…іеҝғе®һзҺ°
  • buildTool() зҡ„й»ҳи®ӨеҖјдҝқиҜҒ fail-closedпјҢж–°е·Ҙе…·ж— жі•йҒ—жјҸе…ій”®е®үе…Ёж–№жі•
  • MCP е·Ҙе…·дёҺеҶ…зҪ®е·Ҙе…·еҜ№жЎҶжһ¶е®Ңе…ЁйҖҸжҳҺпјҢж— йңҖзү№ж®ҠеӨ„зҗҶи·Ҝеҫ„

зјәзӮ№/еұҖйҷҗпјҡ

  • жҺҘеҸЈиҝҮеӨ§пјҲзәҰ 40 дёӘжҲҗе‘ҳпјүпјҢе®һзҺ°е®Ңж•ҙе·Ҙе…·йңҖиҰҒеӨ§йҮҸж ·жқҝд»Јз Ғ
  • жёІжҹ“ж–№жі•пјҲrenderToolUseMessage зӯүпјүдёҺжү§иЎҢйҖ»иҫ‘иҖҰеҗҲеңЁеҗҢдёҖжҺҘеҸЈпјҢиҝқеҸҚе…іжіЁзӮ№еҲҶзҰ»
  • жіӣеһӢеҸӮж•°еўһеҠ дәҶзұ»еһӢзі»з»ҹеӨҚжқӮеәҰпјҢжЎҶжһ¶еұӮжҢҒжңү Tool<AnyObject, unknown> ж—¶дёўеӨұзІҫзЎ®зұ»еһӢ
